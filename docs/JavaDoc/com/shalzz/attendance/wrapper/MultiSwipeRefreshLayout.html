      • canChildScrollUp

        public boolean canChildScrollUp()
        This method controls when the swipe-to-refresh gesture is triggered. By returning false here we are signifying that the view is in a state where a refresh gesture can start.

        As SwipeRefreshLayout only supports one direct child by default, we need to manually iterate through our swipeable children to see if any are in a state to trigger the gesture. If so we return false to start the gesture.

        Overrides:
        canChildScrollUp in class android.support.v4.widget.SwipeRefreshLayout
        Returns:
        Whether it is possible for the child view of this layout to scroll up. Override this if the child view is a custom view.
